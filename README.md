# HappyIconDLL

Windows 애플리케이션용 아이콘 리소스 DLL입니다. 200개의 아이콘을 하나의 DLL 파일에 담아 제공합니다.

## 미리보기

![HappyIcon Preview](preview.png)

## 사용 방법

1. `HappyIcon.dll` 파일을 프로그램 실행 파일(.exe)과 같은 폴더에 넣습니다.
2. 프로그램에서 아이콘 리소스 ID를 지정하여 아이콘을 불러옵니다.
